# 네이버 웹툰 북마크 크롬 익스텐션
웹툰 찾는 시간도 이제 아깝다 !! <br>
미리 등록해두고 메인에서 바로바로 보자 !

# 스택
- React
- Typescript
- Chakra UI
- CRXJS - https://crxjs.dev/vite-plugin

# 버그
- 익스텐션 버튼으로 북마크 전환할때 북마크 버튼이 바뀌지 않는 문제
- 뒤로가기나 다른 페이지에서 들어올때 익스텐션이 작동하지 않는 문제
